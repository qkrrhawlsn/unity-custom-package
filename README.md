# unity-custom-package

### 🎮 패키지 핵심 기능 개요
- **콘텐츠 컨셉**: 현대인의 스트레스 해소를 위한 VR 공간 콘텐츠 ('레이지 룸' 벤치마킹)
- **타깃 유저**: 스트레스 해소가 필요한 20대 대상
- **핵심 메커니즘**: 오큘러스 리프트 컨트롤러를 활용한 물리 기반의 버튼 입력, 그랩(Grab), 스윙(Swing) 액팅 및 오브젝트 파괴(부서짐) 기능 구현

### 🛠️ 개발 및 호환 환경
- **최적화 버전**: Unity 2020 LTS / 2021 LTS (2021년 11월 빌드 기반)
- **사용 SDK**: Oculus SDK (Oculus Rift 기준 인터랙션 구현)
- **확인 방법**: 빈 Unity 2021 LTS 프로젝트 생성 후, 본 패키지를 `Import Package` 하여 컴파일 및 스크립트 구조를 확인하실 수 있습니다. (※ 현재 실장비가 없는 환경인 관계로 런타임 플레이 시 오큘러스 하드웨어 연결이 필요할 수 있습니다.)

### 📄 주요 스크립트 명칭
- **OnTriggerEnter**: 파편 구현 함수
- **S_Col**: 충돌 후 발생한 파편을 제거하는 함수
